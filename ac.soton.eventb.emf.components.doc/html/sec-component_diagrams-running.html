3 Running the Tool

CODA is an extension of the Rodin Platform. See the Rodin User Handbook for a full description of how to run the Rodin Platform and how to create and verify Event-B models.

Event-B models (including those with CODA models) can be imported and exported from the Rodin platform. Right-click anywhere in the Event-B Explorer view and select Import from the pop-up menu. From the select window (Figure 2), select General - Existing Projects into Workspace and click on Next. (DO NOT select Archive File, even if you are about to import a zip file).

Figure 2 - Importing a Model Project

At the next window, select Select archive file and browse to your archived file. Any projects in the archive will be listed for selection in the centre pane of this window. (Make sure that copy projects into workspace is checked otherwise any changes will be made to the original copy). Click on Finish.

To export models, select the project(s) that you want to archive and choose Export from the pop-up menu. From the window that appears, select General-Archive File and press Next. Type in or browse to a location for the archive file and press Finish (Figure 3). Figure 3 - Exporting a Model Project
